# Production Deployment Checklist - V-Tech Kitchen
# Use this checklist to ensure all features are properly deployed

## Pre-Deployment - Environment Variables

Backend (Render/API server):
- NODE_ENV=production
- MONGO_URI=your-mongodb-connection
- FRONTEND_URL=https://vtechkitchen.com
- JWT_SECRET=your-secure-secret
- MAILERSEND_API_KEY=your-mailersend-key

Frontend (Vercel):
- VITE_API_URL=https://api.vtechkitchen.com

## Backend Deployment Steps

1. Verify node-cron is installed (v4.2.1)
2. Check server logs for: "Cron jobs scheduled"
3. Verify MongoDB indexes are created
4. Test API endpoints with curl

## Frontend Deployment Steps

1. Generate PWA icons (192x192, 512x512)
2. Place icons in public/icons/
3. Verify manifest.webmanifest is accessible
4. Test service worker registration
5. Verify sitemap.xml and robots.txt work

## Google Search Console

1. Verify ownership (file already exists)
2. Submit sitemap: https://vtechkitchen.com/sitemap.xml
3. Monitor for errors (should be 0)

## Feature Testing

Test all 12 features:
- Loyalty Points (dashboard)
- Newsletter (subscribe form)
- GDPR (export/delete data)
- Abandoned Cart (tracking)
- PWA (install prompt)
- Inventory Alerts (low stock emails)

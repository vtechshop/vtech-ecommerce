## 📝 Typography

### Font Families

**Current (System Default):**
```
sans-serif: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto,
            "Helvetica Neue", Arial, sans-serif
```

**Recommended (Custom):**
```
Headings:   font-display → "Poppins", sans-serif
Body:       font-sans    → "Inter", sans-serif
Code/Data:  font-mono    → "JetBrains Mono", monospace
```

### Font Sizes & Line Heights

```
text-xs     12px / 16px    ├─ Badges, labels, tiny helper text
text-sm     14px / 20px    ├─ Secondary text, form labels
text-base   16px / 24px    ├─ Body text, paragraph default
text-lg     18px / 28px    ├─ Emphasized text, prices
text-xl     20px / 28px    ├─ Card headings, subtitles
text-2xl    24px / 32px    ├─ Section headings
text-3xl    30px / 36px    ├─ Page titles
text-4xl    36px / 40px    ├─ Hero headings
text-5xl    48px / 1       ├─ Large display text
text-6xl    60px / 1       └─ Extra large display
```

### Font Weights

```
font-normal     400  ├─ Body text, paragraphs
font-medium     500  ├─ Labels, emphasized text (MOST USED)
font-semibold   600  ├─ Headings, card titles
font-bold       700  └─ Page headings, prices
```

## 🎭 Animations

### Hover Effects

```css
/* Card Hover */
hover:-translate-y-1 hover:shadow-2xl

/* Button Hover */
hover:scale-105 active:scale-95

/* Link Hover */
hover:text-primary-700 transition-colors duration-300
```

### Transitions

```css
/* Standard */
transition-all duration-300

/* Slow */
transition-all duration-500

/* Fast */
transition-all duration-200
```

### Keyframe Animations

```
fadeIn          ← Fade in from transparent
fadeInUp        ← Slide up + fade in
fadeInDown      ← Slide down + fade in
slideInRight    ← Slide from right
slideInLeft     ← Slide from left
scaleIn         ← Scale from small to normal
float           ← Gentle floating motion
pulse           ← Pulsing scale
shimmer         ← Shimmer/shine effect
```

## 📱 Responsive Design

### Breakpoints

```
sm:   640px   ├─ Small tablets
md:   768px   ├─ Tablets (MOST COMMON)
lg:   1024px  ├─ Desktop (COMMON)
xl:   1280px  ├─ Large desktop
2xl:  1536px  └─ Extra large desktop
```

### Common Responsive Patterns

```html
<!-- Grid: 1 col mobile, 3 cols tablet, 4 cols desktop -->
<div class="grid grid-cols-1 md:grid-cols-3 lg:grid-cols-4 gap-6">

<!-- Text size: larger on desktop -->
<h1 class="text-3xl md:text-4xl lg:text-5xl font-bold">

<!-- Padding: more on larger screens -->
<section class="py-8 md:py-12 lg:py-16">

<!-- Hide on mobile, show on tablet+ -->
<div class="hidden md:block">
```

## ✅ Accessibility

### Color Contrast Ratios (WCAG AA)

```
✓ text-gray-900 on white        = 16.1:1  (AAA)
✓ text-gray-700 on white        = 8.6:1   (AAA)
✓ text-gray-600 on white        = 5.7:1   (AA)
✓ text-primary-600 on white     = 4.9:1   (AA)
✓ white on bg-primary-600       = 4.5:1   (AA)
✓ white on bg-dark-500          = 19.5:1  (AAA)
```

### Focus States

```css
/* All interactive elements should have visible focus */
focus:ring-2 focus:ring-primary-500 focus:ring-offset-2 focus:outline-none
```

### Font Sizes

```
✓ Minimum body text:     16px (text-base)
✓ Minimum small text:    14px (text-sm)
✓ Use relative units when possible
```
